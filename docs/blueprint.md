# **App Name**: ScoreFlow

## Core Features:

- Admin Secure Access: Provide a secure login system for administrators to access a private dashboard for all tournament management tasks.
- Tournament Lifecycle Management: Administrators can create new tournaments, define formats (e.g., league, knockout), set dates, and manage their status (active, completed).
- Team and Player Registry: Administrators can register and manage teams and individual players participating in each tournament.
- Match Scheduling & Result Entry: Administrators can schedule matches, input scores in real-time or post-game, and update match results.
- Public Tournament Overview: A public-facing page displaying a list of all active and upcoming tournaments for general browsing.
- Public Tournament Deep Dive: Detailed public pages for each tournament, showing teams, full schedules, results, and current standings.
- AI Tournament Highlights Tool: Generate concise textual summaries of completed tournaments, highlighting key results, standout teams, and dramatic moments for public viewing.

## Style Guidelines:

- Primary color: A professional and stable blue (#336699) to convey reliability and expertise. RGB: (51, 102, 153).
- Background color: A very light, desaturated blue (#EEF1F4) to create a clean, airy, and calming backdrop. RGB: (238, 241, 244).
- Accent color: A vibrant, clear violet-blue (#6666E6) used for interactive elements and key information, providing good contrast. RGB: (102, 102, 230).
- Body and headline font: 'Inter', a grotesque-style sans-serif for its modern, clean, and objective aesthetic, suitable for all content.
- Use minimalist, line-art icons that maintain a clean and modern feel, specifically related to sports and data management (e.g., soccer ball, trophy, calendar, gears).
- Employ a responsive, grid-based layout with generous whitespace to ensure readability and clear information hierarchy across various devices. The layout should be intuitive and straightforward.
- Incorporate subtle and smooth transition animations for navigation and state changes, enhancing the user experience without being distracting or delaying interactions.
